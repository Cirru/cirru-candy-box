
Cirru Candy Box
------

Demo: http://repo.cirru.org/candy-box

Candy box shows each value in current scope,
and makes REPL simple and interesting.

Supported commands:

```cirru
set a 1
set b 2
set c $ + a b 4
set d $ - c a b
```

Keyboard events:

* Enter - fun command
* Up - history previous
* Down - history next
